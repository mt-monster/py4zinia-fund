# 基金投资分析平台

这是一个综合性的基金投资分析平台，整合了多种分析工具和策略。

## 项目结构

```
pro2/
├── reports/                          # 📊 统一报告输出目录
│   ├── *.png                        # 分析图表文件
│   ├── *.csv                        # 数据导出文件
│   └── *.md                         # 分析报告
├── fund_search/                     # 🔍 基金搜索分析模块
│   ├── enhanced_main.py             # 主程序入口
│   ├── backtesting/                 # 回测相关
│   │   ├── advanced_strategies.py   # 高级策略库
│   │   └── enhanced_engine/         # 增强回测引擎
│   ├── data_retrieval/              # 数据获取
│   └── shared/                      # 共享配置
├── requirements.txt                 # 依赖包列表
├── consolidate_reports.py           # 目录整合脚本
└── README.md                        # 项目说明
```

## 核心功能

### 🎯 策略对比分析
- **4种高级策略**：双均线动量、均值回归、目标市值、网格交易
- **多维度对比**：收益、风险、夏普比率、胜率等指标
- **智能推荐**：基于风险偏好的策略选择
- **详细报告**：操作建议和投资计划

### 📊 基金分析
- **实时数据分析**：基金净值、收益率、绩效指标
- **趋势判断**：基于动量策略的买卖信号
- **组合分析**：多基金组合的协同效应
- **可视化报告**：图表展示和数据导出

### 🔬 回测引擎
- **历史回测**：长期策略表现验证
- **风险评估**：最大回撤、波动率分析
- **绩效归因**：收益来源和风险因子分析
- **蒙特卡洛模拟**：概率分布和置信区间

## 快速开始

### 环境准备
```bash
pip install -r requirements.txt
```

### 策略对比分析
```bash
# 运行策略对比分析
python fund_search/enhanced_main.py --strategy-analysis

# 自定义参数分析
python fund_search/enhanced_main.py -S \
    --strategy-risk-profile aggressive \
    --strategy-base-amount 2000 \
    --strategy-portfolio-size 10
```

### 基金分析
```bash
# 分析持仓基金
python fund_search/enhanced_main.py --analyze

# 完整分析流程
python fund_search/enhanced_main.py --all
```

- **数据文件**：CSV格式的详细数据导出

## 策略说明

### 双均线动量策略 (Dual MA)
- **适用场景**：趋势明显的市场环境
- **优势**：捕捉中长期趋势，避免假突破
- **参数**：短期均线20日，长期均线60日

### 均值回归策略 (Mean Reversion)
- **适用场景**：震荡整理的市场
- **优势**：低买高卖，逆向投资
- **参数**：均线周期250日，偏离阈值5%

### 目标市值策略 (Target Value)
- **适用场景**：长期定投和稳健投资
- **优势**：成本平均，平滑波动
- **参数**：每月增长目标，动态调整

### 网格交易策略 (Grid Trading)
- **适用场景**：波动率适中的震荡市
- **优势**：分批建仓，降低平均成本
- **参数**：网格间距3%，资金分层

## 注意事项

1. **历史数据限制**：回测基于历史数据，不代表未来表现
2. **市场风险**：投资有风险，建议谨慎决策
3. **参数调优**：策略参数需要根据市场环境调整
4. **资金管理**：严格执行仓位控制和风险管理
